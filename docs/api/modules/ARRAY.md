# Module: lib.array

[README](../../../README.md) -> [API Index](../INDEX.md) -> [Modules Index](./INDEX.md) -> [lib.array](./ARRAY.md)

Source: `src/lib/array.js`

Array coercion and manipulation helpers for list normalization workflows.

## Exported Functions

* [`append`](../functions/array/append.md) - Wrap each item in `input` with a prefix and postfix.
* [`subtract`](../functions/array/subtract.md) - Return a copy of `list` with all values in `exclude` removed.
* [`trim`](../functions/array/trim.md) - Normalize a value into an array and trim whitespace from string elements.
* [`is`](../functions/array/is.md) - Check whether a value is an Array.
* [`to`](../functions/array/to.md) - Coerce any input into an array.
* [`len`](../functions/array/len.md) - Return array length, or 0 if not an array.
* [`filterStrings`](../functions/array/filterStrings.md) - Normalize an input into an array of non-empty strings.
