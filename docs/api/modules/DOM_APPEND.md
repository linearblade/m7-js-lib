# Module: lib.dom.append

[README](../../../README.md) -> [API Index](../INDEX.md) -> [Modules Index](./INDEX.md) -> [lib.dom.append](./DOM_APPEND.md)

Source: `src/lib/dom/append.js`

DOM insertion and replacement helpers with position-based placement.

## Exported Functions

* [`before`](../functions/dom-append/before.md) - Insert `e` before `target`.
* [`after`](../functions/dom-append/after.md) - Insert `e` after `target`.
* [`prepend`](../functions/dom-append/prepend.md) - Insert `e` as first child of `target`.
* [`append`](../functions/dom-append/append.md) - Insert `e` as last child of `target`.
* [`beforeBegin`](../functions/dom-append/beforeBegin.md) - Insert `e` immediately before `target` using adjacent positioning.
* [`afterBegin`](../functions/dom-append/afterBegin.md) - Insert `e` as the first child inside `target`.
* [`beforeEnd`](../functions/dom-append/beforeEnd.md) - Insert `e` as the last child inside `target`.
* [`afterEnd`](../functions/dom-append/afterEnd.md) - Insert `e` immediately after `target` using adjacent positioning.
* [`adjacent`](../functions/dom-append/adjacent.md) - Insert using DOM-standard positions (mirrors insertAdjacentElement).
* [`replace`](../functions/dom-append/replace.md) - Replace target with element.
* [`remove`](../functions/dom-append/remove.md) - Remove target from DOM.
* [`empty`](../functions/dom-append/empty.md) - Empty a target (remove all children).
* [`resolveTarget`](../functions/dom-append/resolveTarget.md) - Resolve a target-ish input into a DOM Element.
