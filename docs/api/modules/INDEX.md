# API Modules Index

[README](../../../README.md) -> [API Index](../INDEX.md) -> [Modules Index](./INDEX.md)

Each module page provides:

* module purpose and source path
* exported functions with short descriptions
* links to individual function reference pages

## Modules

* [lib._boot](./BOOT.md) - Environment bootstrap helpers for root resolution and `lib._env` installation.
* [lib._http](./HTTP.md) - Low-level XHR transport helpers used by bootstrap/runtime internals.
* [lib.args](./ARGS.md) - Arguments-object slicing and positional argument parsing utilities.
* [lib.array](./ARRAY.md) - Array coercion and manipulation helpers for list normalization workflows.
* [lib.bool](./BOOL.md) - Explicit boolean intent detection and conversion helpers.
* [lib.func](./FUNC.md) - Function lookup, wrapping, and composition helpers.
* [lib.hash](./HASH.md) - Plain-object utilities for deep path access, transforms, and filtering.
* [lib.number](./NUMBER.md) - Numeric coercion and bounded conversion helpers.
* [lib.require](./REQUIRE.md) - Dependency-gate helpers for required library paths and services.
* [lib.service](./SERVICE.md) - In-memory service registry with conventional start/stop dispatch.
* [lib.str](./STR.md) - String coercion and lightweight text utility helpers.
* [lib.utils](./UTILS.md) - Cross-module normalization aliases plus shared classification helpers.
* [lib.dom](./DOM.md) - DOM element resolution, attribute/property access, and helper entrypoints.
* [lib.dom.create](./DOM_CREATE.md) - DOM element factory helpers for scripts, styles, and generic nodes.
* [lib.dom.append](./DOM_APPEND.md) - DOM insertion and replacement helpers with position-based placement.
* [lib.dom.form](./DOM_FORM.md) - Form collection and lightweight submit orchestration helpers for DOM forms.
* [lib.dom.transform](./DOM_TRANSFORM.md) - Declarative DOM mapping helpers for subtree and list data transforms.
